# Quick Proxy

```
Usage: npx featherbear/quickproxy [options] <target>

Arguments:
  target             Target endpoint, include http:// or https://

Options:
  -p, --port <port>  Proxy port (default: 0)
  -h, --host <host>  Proxy host (default: "0.0.0.0")
  --help             display help for command
```
